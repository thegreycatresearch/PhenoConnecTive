# Clinical Problem

Genetic panels for connective tissue disorders often produce complex variant data requiring extensive manual interpretation.

Clinical phenotypes are frequently under-structured or poorly integrated into computational analysis pipelines.

This may contribute to:
- delayed diagnosis,
- uncertain variant interpretation,
- incomplete subtype prioritization,
- and reduced genotype-phenotype correlation.

PhenoConnecTive aims to address this challenge through phenotype-guided bioinformatic prioritization.
